# sam-link-copier
Copy SAM.gov opportunity links from ezgovopps.com instantly. One click. No scrolling. No manual selection.
Privacy Policy

This Chrome extension does not collect, store, transmit, or share any personal or sensitive user data.

All processing is performed locally within the user's browser. The extension only interacts with the active tab after the user explicitly clicks the extension icon.

The extension does not:
- Track browsing activity
- Collect personally identifiable information (PII)
- Use cookies or any form of tracking
- Send data to external servers

Permissions Usage:
- activeTab: Used only after user interaction to access the current page
- scripting: Used to analyze page content and extract relevant links

The extension operates only on pages relevant to its functionality and does not perform background data collection.
